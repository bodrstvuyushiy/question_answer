## question_answer

1. Зависимости настроеные через Poetry, можно установить командой:`make deps`.
2. Запуск проекта: `make dev` - собирает образ в контейнере, поднимает БД и web-app, прогоняет линтеры, миграции и тесты.
3. Поднять - `make up` и остановить - `make down` контейнер.
4. Запуск сервера `make run`
В Makefile прописаны все команды.


Только авторизованные пользователи могут писать questions и asnwers. Создаем пользователя в админке и берем под него токен.
Чтобы при отправке POST запроса на answer сохрянался актульаный UUID пользователя, создал дополнительно отдельно accounts приложение для кастомного пользователя.


Логика и технические требования соблюдены.
Логирование на уровне INFO. Базовые тесты есть.
